# Faith Heights Business Fellowship – Alpha Build Pack

This pack contains the current FHBF MVP website preview plus a build-ready Alpha specification for turning FHBF into a year-round Kingdom Business Ecosystem.

## Included
- `index.html` – current frontend preview
- `assets/` – logo and Business Village map assets
- `docs/` – roadmap, user journeys, feature specification and content structure
- `database/` – starter PostgreSQL schema
- `api/` – API route specification
- `wireframes/` – screen-by-screen UI plan

## Core Modules
1. Membership applications and approval
2. Digital membership cards
3. Business directory and member profiles
4. Business Village booking with member/non-member pricing
5. Events and trade missions
6. Opportunities hub
7. Marketplace
8. Sponsor portal
9. Admin dashboard
10. Campus free-access portal concept

## Recommended Stack
- Frontend: Next.js + React + Tailwind
- Backend: NestJS or Express
- Database: PostgreSQL
- Auth: Auth.js / Clerk
- Storage: Cloudflare R2 or S3
- Payments: Stripe/PayPal + manual EcoCash/ZIPIT proof upload
