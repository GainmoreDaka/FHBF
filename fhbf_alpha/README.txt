Faith Heights Business Fellowship MVP Platform v4

Open index.html in a browser.
This is a static preview/demo build. Forms store demo data in browser localStorage and CSV export works locally.
Production integration needed: database, authentication, payment gateway, file upload, email/WhatsApp notifications, admin security.

- Business Village member/non-member price calculator
- Village booking demo
- Sponsor application demo
- Directory preview
- Events, opportunities, missions and campus free-access section
- Digital membership card preview
- Admin metrics and CSV export
