# FHBF Omega Plus Working MVP

This package is a lightweight working MVP prototype for Faith Heights Business Fellowship.

It includes:
- Public pages
- Membership application form
- Business Village booking form
- Member vs non-member pricing
- Admin preview dashboard
- Local JSON-based demo storage
- Directory and opportunities placeholder pages

## Run locally

```bash
npm install
npm run dev
```

Then open:

```text
http://localhost:3000
```

## Important

This is still a prototype. For production, replace JSON storage with PostgreSQL + Prisma, add authentication, file uploads, payment confirmation and real email/WhatsApp notifications.
