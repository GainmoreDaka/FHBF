# Business Village Workflow

## Stand Types
- Emerging Entrepreneur
- Standard Indoor Stand
- Premium Indoor Stand
- Outdoor Display
- Vehicle Display
- Equipment Display

## Pricing Logic
If user has active FHBF membership, apply member price.
If not active, apply non-member price.

## Example Pricing
Emerging Entrepreneur: Member $30 / Non-Member $60
Standard Indoor: Member $100 / Non-Member $150
Premium Indoor: Member $150 / Non-Member $250
Outdoor Display: Member $150 / Non-Member $250
Vehicle Display: Member $200 / Non-Member $350
Equipment Display: Member $250 / Non-Member $400

## Booking Flow
1. User selects stand type.
2. User chooses available stand or display area.
3. System calculates price.
4. User submits booking.
5. Stand status becomes Reserved.
6. User pays or uploads proof.
7. Admin verifies payment.
8. Stand status becomes Booked.
9. Confirmation and QR exhibitor pass are issued.
