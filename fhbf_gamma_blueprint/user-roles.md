# User Roles & Permissions

## Visitor
- Browse public website
- View public directory
- Register for public events

## Applicant
- Submit membership application
- Upload payment proof
- Track application status

## Member
- Manage personal profile
- View member-only content
- Access events, opportunities, and marketplace
- Download digital membership card

## Business Member
- Manage business profile
- Post marketplace listings
- Post opportunities
- Book Business Village stands at member price

## Corporate Partner
- Manage company profile
- Post tenders / RFQs
- Access sponsor tools
- Add staff users

## Sponsor
- Manage sponsorship package
- Upload branding assets
- Download invoices

## Event Admin
- Manage registrations
- Scan QR codes
- Export attendance

## Chapter Admin
- Manage members, events, and businesses for assigned chapter

## Super Admin
- Full platform access
- System settings
- Billing, reports, users, roles
