# Membership Workflow

1. User selects membership level.
2. User submits application.
3. User uploads proof of payment or pays online.
4. Application status becomes Pending Review.
5. Admin reviews application.
6. Admin can approve, reject, or request more information.
7. On approval:
   - Membership number is generated.
   - Digital QR membership card is generated.
   - Member dashboard is activated.
   - Business directory profile is created if consent was given.
   - Welcome email/WhatsApp is sent.

## Membership Number Format
FHBF-YYYY-000001

## Membership Levels
- Student: US$20/year
- Entrepreneur: US$50/year
- Professional: US$100/year
- Business: US$250/year
- Corporate Partner: US$500+/year
