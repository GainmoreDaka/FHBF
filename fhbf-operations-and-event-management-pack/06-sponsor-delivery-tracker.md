# Sponsor Delivery Tracker

## Purpose
To make sure every sponsor receives promised benefits.

## Track
- Sponsor name
- Package level
- Amount
- Contact person
- Logo received
- Payment status
- Website listing
- Social media feature
- Event signage
- Speaking opportunity
- VIP invitations
- Stand allocation
- Benefits delivered
- Renewal date

## Benefit Status
- Not started
- In progress
- Delivered
- Not applicable

## Post Event
Send sponsor report including:
- Photos
- Attendance estimate
- Media reach
- Branding delivered
- Thank you message
- Renewal invitation
