# Exhibitor Management Workflow

## Workflow
Application → Review → Approval → Payment → Stand Allocation → Setup Information → Check-In → Exhibition → Follow-Up

## Application Data
- Business name
- Contact person
- Phone/WhatsApp
- Email
- Category
- Products/services
- Stand type
- Electricity needs
- Table/chair needs
- Vehicle/equipment needs
- Payment proof

## Admin Statuses
- New application
- Approved
- Awaiting payment
- Paid
- Stand allocated
- Setup information sent
- Checked in
- Followed up

## Exhibitor Communication
- Application received
- Payment request
- Stand allocation
- Setup instructions
- Event reminder
- Thank you and feedback request
