# Visitor Experience Framework

## Desired Visitor Journey
Welcome → Discover → Connect → Engage → Follow Up

## Welcome
Visitors should immediately understand:
- Where to enter
- Where to register
- What is happening
- Where to go next

## Discover
Visitors should be able to find:
- Main marquee
- Exhibitors
- Vehicle displays
- Sponsors
- Networking garden
- Hospitality
- First aid
- Toilets

## Connect
Provide:
- Directory QR code
- Exhibitor QR codes
- WhatsApp links
- Business cards
- Networking spaces

## Follow-Up
After event:
- Thank you message
- Directory link
- Membership invitation
- Sponsor information
- Next event invitation
