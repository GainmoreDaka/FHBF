# Admin Workflow

Daily tasks:
1. Open FHBF Platform Data Google Sheet.
2. Check new membership applications.
3. Check new Business Village bookings.
4. Check sponsor enquiries.
5. Update status column.
6. Send WhatsApp/email follow-ups.

Status values:
Membership: Submitted, Under Review, Awaiting Payment, Approved, Rejected, More Info Needed.
Business Village: Submitted, Awaiting Payment, Paid, Approved, Stand Allocated, Cancelled.
Sponsors: New Lead, Contacted, Proposal Sent, Confirmed, Paid, Benefits Delivered.

Membership number format: FHBF-2026-000001
