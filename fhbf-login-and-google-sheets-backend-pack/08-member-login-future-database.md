# Future Full Login System

When FHBF moves beyond Google Sheets, build:

## Database Tables
- users
- roles
- permissions
- memberships
- membership_applications
- businesses
- village_bookings
- payments
- sponsors
- events
- notifications

## Authentication
- Email and password
- Password reset
- Email verification
- Role-based dashboard

## Member Login
Members can:
- Edit profile
- View card
- Renew membership
- Book Village stand
- Register for events
- View opportunities

## Admin Login
Admins can:
- Approve members
- Confirm payments
- Manage stands
- Export reports
- Send notifications
