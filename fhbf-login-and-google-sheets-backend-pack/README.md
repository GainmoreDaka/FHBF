# FHBF Login + Google Sheets Backend Pack

This pack explains how to move the current FHBF page from a browser-only demo into a working live system using Google Sheets first, then a full database later.

Goal:
- Public website remains visible.
- Membership applications save to Google Sheets.
- Business Village bookings save to Google Sheets.
- Sponsor registrations save to Google Sheets.
- Contact messages save to Google Sheets.
- Admin can view applications in Google Sheets.
- Member/Admin login pages are planned and added to the website structure.

This is the fastest bridge between the current page and a proper FHBF platform.
