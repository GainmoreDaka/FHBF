# Login Pages Plan

## Member Login Page

URL:
/fhbf/login

Fields:
- Email
- Password

Buttons:
- Login
- Forgot Password
- Apply for Membership

## Member Dashboard

URL:
/fhbf/member

Shows:
- Membership status
- Membership number
- Digital card
- Business profile
- Business Village bookings
- Events
- Opportunities

## Admin Login

URL:
/fhbf/admin-login

For security, do not make this too prominent.

## Admin Dashboard

URL:
/fhbf/admin

Shows:
- Membership applications
- Village bookings
- Sponsors
- Contact messages
- Payments
- Reports

## First Stage Recommendation

Use Google Sheets first.

Admin login can simply be password-protected or hidden while the Google Sheet acts as the real admin database.

Later, build a proper login system with:
- Email/password
- Role-based permissions
- Database
- Secure sessions
