# Google Sheets Backend Setup

Create one Google Sheet named:

FHBF Platform Data

Add the following tabs:

## 1. Membership Applications
Columns:
- Timestamp
- Reference
- Full Name
- Email
- Phone
- Membership Type
- Business Name
- Industry
- Seven Mountain
- Province / City
- Directory Consent
- Payment Reference
- Status
- Admin Notes

## 2. Business Village Bookings
Columns:
- Timestamp
- Reference
- Applicant Name
- Business Name
- Email
- Phone
- Is Member
- Membership Number
- Space Type
- Amount Due
- Payment Reference
- Status
- Stand Number
- Admin Notes

## 3. Sponsors
Columns:
- Timestamp
- Reference
- Organisation
- Contact Person
- Email
- Phone
- Sponsorship Level
- Notes
- Status
- Payment Reference
- Admin Notes

## 4. Contact Messages
Columns:
- Timestamp
- Reference
- Name
- Email
- Phone
- Message
- Status
- Admin Notes

## 5. Members
Columns:
- Membership Number
- Full Name
- Email
- Phone
- Membership Type
- Business Name
- Start Date
- Expiry Date
- Status
- Directory Profile URL
