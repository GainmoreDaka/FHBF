# Admin Workflow Using Google Sheets

## Membership Applications

Admin reviews:
- Applicant details
- Membership type
- Payment reference
- Directory consent

Admin changes status:
- Submitted
- Under Review
- Approved
- Rejected
- More Info Needed

When approved:
1. Generate membership number.
2. Add to Members tab.
3. Send welcome message.
4. Create directory listing.
5. Issue digital card.

## Business Village Bookings

Admin reviews:
- Space type
- Member/non-member
- Amount due
- Payment reference

Admin changes status:
- Submitted
- Awaiting Payment
- Paid
- Approved
- Stand Allocated

Admin adds:
- Stand number
- Notes

## Sponsors

Admin reviews:
- Sponsorship level
- Contact person
- Payment status
- Benefits to deliver

Admin changes status:
- Lead
- Contacted
- Confirmed
- Paid
- Benefits Delivered
