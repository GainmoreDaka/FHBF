# Admin Workflows

## Membership Approval

1. Application submitted
2. Admin reviews details
3. Admin checks payment proof
4. Admin approves
5. Membership number generated
6. Digital card created
7. Welcome notification sent
8. Directory profile activated

## Business Village

1. Exhibitor applies
2. Admin reviews category
3. Stand allocated
4. Payment confirmed
5. Exhibitor pack sent
6. QR pass generated

## Directory

1. Member submits business profile
2. Admin reviews content
3. Admin approves listing
4. Profile becomes public

## Events

1. Event created
2. Tickets/pricing added
3. Registration opens
4. QR confirmations sent
5. Attendance scanned
6. Report generated
