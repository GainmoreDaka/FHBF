# FHBF Authentication & Security Model

## Login Methods
Phase 1: Email and password.  
Phase 2: Phone/WhatsApp OTP, Google login and Microsoft login.

## Security
- Email verification
- Password reset
- Login audit trail
- Session expiry
- Role-based access control
- Admin activity logs

## Sensitive Actions
Require confirmation for approving membership, deleting members, confirming payments, changing roles, issuing refunds and exporting financial reports.

## Admin Security
Super Admin accounts should be limited and protected with strong passwords and two-factor authentication when available.
