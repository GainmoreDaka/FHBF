# User Journeys

## Membership Application
Visitor selects membership level → completes personal and business details → consents to directory listing → uploads proof of payment → receives application reference → admin reviews → approval generates membership number, digital card and directory profile.

## Business Village Booking
User selects member/non-member → chooses stand type → sees price → completes exhibitor details → uploads logo/proof of payment → admin allocates stand → user receives confirmation.

## Member Directory
Member logs in → updates business profile → uploads logo, cover photo, services, products and contacts → admin verifies → listing becomes public.

## Trade Mission
Member views upcoming mission → registers interest → uploads documents → pays deposit → downloads itinerary → receives WhatsApp/email updates.

## Campus Free Access
Visitor connects to campus Wi-Fi → captive portal opens FHBF homepage → free pages include homepage, village map, directory, schedule, registration and emergency contacts → general internet requires voucher/login.
