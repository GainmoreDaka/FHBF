# MVP User Journeys

## New Member
1. Visits Membership page
2. Chooses membership level
3. Completes application
4. Uploads proof of payment
5. Receives application reference
6. Admin approves
7. Membership number generated
8. Digital membership card issued
9. Business profile created

## Business Village Exhibitor
1. Visits Business Village page
2. Selects stand type
3. Chooses member/non-member
4. Enters membership number if member
5. Price calculated
6. Uploads payment proof
7. Admin allocates stand
8. Confirmation sent

## Directory Visitor
1. Searches by category/location
2. Opens business profile
3. Clicks WhatsApp/email/request quote
4. Lead is recorded

## Admin
1. Logs in
2. Reviews pending applications
3. Confirms payment
4. Approves/rejects
5. Exports reports
